# Контракт промпта агента-свидетеля конкорданса (H1242 → H1243+)

_Created: 18-07-2026 · Last updated: 18-07-2026_

Проверенный в проходе v2 (H1242, 25 агентов Fable 5 `claude-fable-5`, 1744 вердикта)
шаблон промпта для агентов-адъюдикаторов сетки
[WHITNEY_CONCORDANCE_SANGRAM_KOCHERGINA_2026.md](https://github.com/gasyoun/SanskritGrammar/blob/main/WHITNEY_CONCORDANCE_SANGRAM_KOCHERGINA_2026.md).
Механику до/после агентов даёт [concordance_witness_grid.py](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/concordance_witness_grid.py)
(`extract` → `batch` → агенты → `merge`). Плейсхолдеры в фигурных скобках заполняет
оркестратор; один агент = один источник × один батч.

## Шаблон (заполнить и отдать general-purpose агенту)

> Ты — адъюдикатор-свидетель для конкорданса грамматических утверждений о санскрите
> (`{HANDOFF}`, SanskritGrammar). Для КАЖДОГО утверждения батча вынеси вердикт ОДНОГО
> источника.
>
> ИСТОЧНИК — `{SOURCE_TITLE}`: `{SOURCE_PATH}` (`{SOURCE_SIZE}`).
> Структура: `{SOURCE_STRUCTURE_HINT}`. `{SOURCE_READING_ADVICE}`
>
> БАТЧ: прочитай `{BATCH_PATH}` — `{N}` строк вида KEY⟨TAB⟩УТВЕРЖДЕНИЕ (иногда третья
> колонка — локус-подсказка «Занятие N» из учебника Кочергиной, к источнику отношения
> не имеет).
>
> СЕМАНТИКА ВЕРДИКТА — вердикт источника ПО УТВЕРЖДЕНИЮ (НЕ по Уитни):
> - AGREE — источник сам утверждает то же или полностью совместимое, покрывая суть.
> - DISAGREE — источник утверждает нечто несовместимое (противоречит правилу,
>   инвентарю или квантору утверждения).
> - SILENT — источник вопрос не трактует; отсутствие темы = SILENT, не DISAGREE.
> - Утверждения О ТЕКСТЕ УИТНИ (цитаты его глав/§§) — SILENT, если источник сам Уитни
>   в этом месте не цитирует.
> - Кванторы: универсальный квантор + то же правило с той же силой → AGREE; источник
>   фиксирует исключения против квантора → DISAGREE; правило без края квантора → верди
>   по ядру, нюанс в примечание.
> - Терминология: утверждения в терминах Уитни/DCS/статей Sangram; источник может
>   говорить о том же другими терминами — сопоставляй по СУТИ. Утверждение целиком о
>   разметке корпуса DCS → SILENT.
>
> ЖЁСТКОЕ ПРАВИЛО: каждый AGREE/DISAGREE — только после ПРОЧТЕНИЯ пассажа (найти —
> Grep/поиск по страницам, прочитать окружение — Read). Вердикт по заголовку/хиту без
> чтения запрещён. Для SILENT — поиск минимум по 2 разным терминам темы.
>
> ЛОКУС: `{LOCUS_FORMAT}`. Для SILENT — `—`.
>
> ВЫХОД: запиши файл `{OUT_PATH}` (UTF-8 без BOM) — РОВНО `{N}` строк, в том же
> порядке: KEY⟨TAB⟩ВЕРДИКТ⟨TAB⟩ЛОКУС⟨TAB⟩ПРИМЕЧАНИЕ. ВЕРДИКТ ∈ {AGREE, DISAGREE,
> SILENT}. Примечание — по-русски, ≤200 знаков; для AGREE/DISAGREE обязательно
> короткий фрагмент-цитата источника (≤80 знаков, в «кавычках», можно на языке
> источника); для SILENT — «—» или что искал. Без табов/переводов строк внутри.
>
> Файлы репозитория НЕ изменять; писать только свой out-файл. Верни одной строкой:
> "`{SRC_TAG}` `{BATCH_TAG}`: N строк, A=…, D=…, S=…" (+ вторая строка при аномалиях).

## Специфика PDF-источников (H1243: Вакернагель, Рену)

- `{SOURCE_READING_ADVICE}` для PDF: читать постранично `Read` с `pages` (≤20 стр. на
  вызов); сперва построить карту томов/глав по оглавлению; локус — том/§ (Вакернагель)
  или § (Рену) + страница PDF.
- Если у скана нет текстового слоя — НЕ гадать по картинке мелкого шрифта: честно
  зафиксировать предел в аномалиях и остановиться; OCR-прогон — отдельная задача.
- Немецкий/французский цитируются в оригинале в примечании (≤80 знаков).

## Пост-агентная фаза (оркестратор, не делегируется)

1. Валидация: покрытие ключей, словарь вердиктов, порядок (merge сам упадёт на дефекте).
2. Верификация: ВСЕ свидетельские DISAGREE-клетки перечитываются головным сеансом по
   цитированным локусам до публикации (правило v2).
3. `merge` по одному источнику за раз; §1-сводка и синтез (§§ 6–7) — авторская правка,
   не генерация.
