# Склонение: местоимения

_Создано: 16-07-2026 · Последнее обновление: 16-07-2026_

> **Статус: статья-кандидат** (слот ядра `SG-MO-010` W2, реестр
> [C2](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/toc/data/articles.json);
> манифест `art:pronouns`). Публикация гейтится авторской визой — до неё все
> формулировки ниже предварительны.

## 1. Класс: местоимения

Местоимения — **закрытый** класс: их конечный, небольшой список не пополняется
(в отличие от открытого именного инвентаря). Разряды:

- **личные** — 1 л. `aham` «я» (косвенная основа `mad`/мн. `asmad`), 2 л. `tvam`
  «ты» (`tvad`/`yuṣmad`); 3-е лицо особого местоимения не имеет — его роль играет
  указательное `tad`;
- **указательные** — `tad` «тот», `idam` / `etad` «этот», `adas` «оный»;
- **вопросительное** — `ka` «кто/что» (Nom. `kaḥ`/`kim`);
- **относительное** — `yad` «который»;
- **неопределённые** — `kaścit`, `kaḥcana` и др. (вопросительное + частица);
- **местоимённые прилагательные** — `sarva` «весь», `anya` «другой», `para`, `sama`
  (склоняются по местоимённому образцу, но имеют род).

Определяющая черта — **супплетивизм и особые окончания**: одна лемма выступает в
разных основах (`aham` в Nom., но `mā-`/`mad-` в косвенных; `sa`/`sā`/`tat` в Nom.,
но `ta-` в косвенных), а ряд окончаний уникален для местоимений (Dat. ед. `-smai`,
Abl. ед. `-smāt`, Loc. ед. `-smin`, Gen. мн. `-ṣām`). Три слоя источников:

| Слой | Источник | Что даёт |
|---|---|---|
| **Аттестовано** | пинованный снапшот DCS (§ 2), `upos=PRON` | покрытие падеж×число на лемму — **исключение из EM3** (§ 3) |
| **Традиционно** | [Уитни 1889, гл. VII §§ 491–521](https://en.wikisource.org/wiki/Sanskrit_Grammar_%28Whitney%29/Chapter_VII); Зализняк 1975 | канонические супплетивные парадигмы (§ 4) |
| **Порождено** | — | для супплетивного закрытого класса **порождение ≡ традиция**: продуктивного правила нет, канонический список и есть эталон |

## 3. Аттестация — исключение из EM3

Здесь корпус ведёт себя **противоположно** склонению существительных. Именной
инвентарь (актив G2, 57 144 леммы) даёт **медиану 1** аттестованную клетку на лемму —
«полная парадигма» почти никогда не встречается (предел EM3). Местоимения — **закрытый
высокочастотный класс из 38 лемм**, и они аттестуют **медиану 12,5** клетки из 24
(среднее 12,13; 39,5 % лемм — ≥ 18 клеток). Причина прямая: 38 лемм несут 544 999
токенов (в среднем ≈ 14 300 на лемму против ≈ 31 у существительного), а высокая частота
вытягивает почти всю парадигму в аттестацию.

| Местоимение | Разряд | Токенов | Клеток /24 |
|---|---|---:|---:|
| `tad` (тот) | указат. | 184 809 | 21 |
| `yad` (который) | относит. | 61 421 | 21 |
| `mad` (я) | личн. 1 л. | 60 575 | 21 |
| `idam` (этот) | указат. | 52 461 | **22** |
| `tvad` (ты) | личн. 2 л. | 49 725 | 21 |
| `etad` (этот) | указат. | 40 991 | 21 |
| `sarva` (весь) | местоим. прил. | 22 865 | 18 |
| `ka` (кто) | вопросит. | 14 087 | 18 |

Практическое следствие для домена: по местоимениям количественные утверждения о
клетках **надёжны** (в отличие от существительных) — но лишь по этим немногим
высокочастотным леммам; редкие клетки (дв. ч., Voc.) и здесь малочисленны.

## 4. Парадигма (традиционно)

Для супплетивного закрытого класса эталон — канонический список
([Уитни гл. VII](https://en.wikisource.org/wiki/Sanskrit_Grammar_%28Whitney%29/Chapter_VII)),
а не порождающее правило. **1-е лицо `aham` «я»** (без рода) — чистый супплетивизм
(Nom. `aham`, но косвенная основа `ma-`/`asma-`):

| | ед. | дв. | мн. |
|---|---|---|---|
| **Nom** | aham | āvām | vayam |
| **Acc** | mām | āvām | asmān |
| **Instr** | mayā | āvābhyām | asmābhiḥ |
| **Dat** | mahyam | āvābhyām | asmabhyam |
| **Abl** | mat | āvābhyām | asmat |
| **Gen** | mama | āvayoḥ | asmākam |
| **Loc** | mayi | āvayoḥ | asmāsu |

**Указательное `tad` «тот» (муж.)** — Nom. `saḥ` без `t`, косвенные на `ta-`, с
местоимёнными окончаниями `-smai`/`-smāt`/`-smin`/`-ṣām`:

| | ед. | дв. | мн. |
|---|---|---|---|
| **Nom** | saḥ | tau | te |
| **Acc** | tam | tau | tān |
| **Instr** | tena | tābhyām | taiḥ |
| **Dat** | tasmai | tābhyām | tebhyaḥ |
| **Abl** | tasmāt | tābhyām | tebhyaḥ |
| **Gen** | tasya | tayoḥ | teṣām |
| **Loc** | tasmin | tayoḥ | teṣu |

(ср. род: Nom./Acc. `tat` / `te` / `tāni`; жен. род: `sā` / `te` / `tāḥ`.)

## 6. Ограничения

1. **Исключение из EM3, а не отмена.** Медиана высока (12,5), но покрытие полное
   лишь у немногих высокочастотных лемм; редкие клетки (дв. ч., Voc.) и здесь скудны —
   утверждения по ним несут знаменатель.
2. **Супплетивизм и лемматизация по косвенной основе.** DCS лемматизирует личные
   местоимения по косвенной основе (`aham` → лемма `mad`, `vayam` → `mad`); поверхностная
   форма ≠ лемма. Читать таблицу § 3 нужно с учётом этого (лемма `mad` покрывает `aham`,
   `mām`, `mayā`, `vayam` … одной строкой).
3. **Порождение ≡ традиция.** Для закрытого супплетивного класса продуктивного
   генератора нет; парадигмы § 4 — канонические (Уитни гл. VII), не машинно-порождённые.
4. **Род у указательных.** `tad`/`idam`/`etad` склоняются по родам (`saḥ`/`sā`/`tat`);
   покрытие § 3 — на лемму по всем родам, без разбивки по роду.
5. **Пин зеркала.** Коммит-пин `04e0778…` осиротел; привязка — provenance-таблица +
   SHA-256 + тег `c3-pin-04e0778-content` (§ 2), как у пилотов P1–P5 и актива G2.
