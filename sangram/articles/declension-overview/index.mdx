# Склонение: категории и обзор системы

_Создано: 16-07-2026 · Последнее обновление: 16-07-2026_

> **Статус: статья-кандидат** (обзорный слот `SG-MO-001` ядра W2, реестр
> [C2](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/toc/data/articles.json);
> манифест `art:declension-overview`). Публикация гейтится авторской визой — до неё
> все формулировки ниже предварительны.

## 1. Рамка кластера

Слот [SG-MO-001](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/toc/03-formal-morphology.mdx)
открывает домен **формальной морфологии** обзором кластера «Склонение». Это
**обзорная статья**, а не пилот: она не проверяет предел свидетельств (kill-gate
отсутствует), а задаёт **структурную и частотную рамку** домена, к которой
привязаны подстатьи SG-MO-002…011.

Санскритское именное словоизменение — это сетка **8 падежей** (Nom, Voc, Acc,
Ins, Dat, Abl, Gen, Loc) × **3 числа** (ед., дв., мн.) × **3 рода** (м., ж., ср.),
реализуемая набором окончаний, распределённых по **типам основ** (по конечному
звуку основы). Традиция ([Уитни 1889, гл. IV, §§ 261–320](https://en.wikisource.org/wiki/Sanskrit_Grammar_%28Whitney%29/Chapter_IV))
описывает категории и схему окончаний; кластер раскладывает основы на подстатьи
(§ «Карта подстатей»). Три слоя источников не смешиваются:

| Слой | Источник | Что даёт |
|---|---|---|
| **Традиционно** | [Уитни 1889, гл. IV](https://en.wikisource.org/wiki/Sanskrit_Grammar_%28Whitney%29/Chapter_IV) §§ 261–320; Кочергина (уроки склонения); Зализняк 1978 (очерк, именное склонение) | категории падеж/число/род, схема окончаний, типы основ |
| **Аттестовано** | пинованный снапшот DCS (§ 2) — словоизменённые именные токены | частотная рамка домена: матрица падеж×число, инвентарь типов основ (§ 3) |
| **Порождено** | парадигма основы любым конъюгатором (MWinflect / vidyut) | полный набор клеток на лемму — но большинство клеток корпусом не аттестовано (предел EM3, пилот P1) |

## 3. Обзор системы

### 3.1 Матрица падеж×число как частотная рамка

Распределение по **числу** сильно скошено к единственному; **двойственное — редкая
категория**:

| Число | Токенов | Доля |
|---|---:|---:|
| ед. (Sing) | 1 419 826 | 79,3 % |
| мн. (Plur) | 332 943 | 18,6 % |
| дв. (Dual) | 37 501 | **2,09 %** [2,07–2,12] |

Распределение по **падежу** — прямые падежи (Nom+Acc) несут почти две трети
употреблений; дательный — самый редкий:

| Падеж | Токенов | Доля |
|---|---:|---:|
| Nom | 692 647 | 38,7 % |
| Acc | 430 396 | 24,0 % |
| Ins | 184 609 | 10,3 % |
| Loc | 177 320 | 9,9 % |
| Gen | 138 883 | 7,8 % |
| Voc | 65 363 | 3,7 % |
| Abl | 61 862 | 3,5 % |
| Dat | 39 190 | **2,2 %** |

Практическое следствие: редкие клетки (дв. число, Dat, Abl) в корпусе
малочисленны, поэтому количественные утверждения по ним требуют явных знаменателей
и ДИ, а покрытие клеток на лемму почти всегда неполно (предел EM3 — доказан пилотом
[P1](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/articles/a-stems/index.mdx)).

### 3.2 Инвентарь типов основ

Токен-взвешенное распределение по типу основы (приближение по финали леммы) —
частотная мотивация порядка подстатей:

| Тип основы | Доля | Слот кластера |
|---|---:|---|
| **-a (тематические)** | **62,9 %** [62,86–63,0] | SG-MO-002 ① |
| -i | 8,9 % | SG-MO-004 ② |
| -ā | 6,4 % | SG-MO-003 ② (ж. р., SG-WF-005) |
| n-основы (-an/-in/-man) | 4,8 % | SG-MO-006/007 |
| s-основы (-as/-is/-us) | 4,2 % | SG-MO-006 ① |
| -ī | 3,6 % | SG-MO-003 ② |
| -u | 3,3 % | SG-MO-004 ② |
| прочие согласные | 2,3 % | SG-MO-006 ① |
| -ū / -ṛ / t-основы / r-основы | ~2 % совокупно | SG-MO-005/006 ② |

**Тематическая основа на -a доминирует (≈ 63 %)** — почему статья -a-основ
(SG-MO-002) выбрана флагманом пилотов и первым ① слотом кластера. Остальные типы
раскладываются по подстатьям § «Карта подстатей».

## 5. Ограничения

1. **Тип основы — приближение по финали леммы.** Классификация идёт по конечному
   звуку цитатной формы леммы, а **цитатная форма ≠ основа** (напр. n-основа rājan
   цитируется на -n, но парадигма имеет сильную/слабую ступени). Доли § 3.2 —
   ориентировочная рамка, а не парадигматический разбор; точное разложение — задача
   подстатей.
2. **Универсум — только нарицательные (`NOUN`).** Прилагательные (`ADJ`),
   местоимения (`PRON`), числительные — отдельные `upos`, вынесены в свои слоты
   (SG-MO-008/009/010/011); нефинальные члены композитов (`Case=Cpd`) исключены.
3. **Полуавтоматическая разметка + известные дефекты C3.** Падеж/число — аннотация
   Хельвига; сандхи и NFD-нормализация (Д3) могут смещать финаль леммы; акцент
   отсутствует. Локальная точность признаков в этой рамке отдельно не измерялась.
4. **Пин зеркала.** Коммит-пин `04e0778…` осиротел после перезаписи истории
   dcs-conllu; действующая привязка — provenance-таблица + SHA-256 + тег
   `c3-pin-04e0778-content` (§ 2), как у пилотов P1–P5.

## 6. Карта подстатей

Кластер «Склонение» раскрывается подстатьями (канон ID — за реестром
[C2](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/toc/03-formal-morphology.mdx);
приоритет ① = ядро W2, ② = расширение):

- **SG-MO-002** основы на -a ① — ✅ опубликовано (пилот P1)
- **SG-MO-003** основы на -ā/-ī/-ū ② · **SG-MO-004** основы на -i/-u ② ·
  **SG-MO-005** основы на -ṛ (агентивно-родственные) ②
- **SG-MO-006** согласные и односложные основы ① · **SG-MO-007** основы на -an/-in ②
- **SG-MO-008** причастные/сравнительные основы -ant/-vāṅs ② · **SG-MO-009** степени
  сравнения ②
- **SG-MO-010** местоимения ① · **SG-MO-011** числительные ②

Обзорная рамка не переизлагает инвентарь слотов и граф зависимостей — они
машинно-генерируются на [странице домена](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/toc/03-formal-morphology.mdx)
из реестра C2 (append-only).
