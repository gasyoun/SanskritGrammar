# Usha Sanka kāraka gold — structured dhātu → kāraka → citation dataset

_Created: 20-07-2026 · Last updated: 20-07-2026_

A structured extraction of the **kāraka-demand model** from Usha Sanka's 2014–15 PhD
*"A Study Of Kāraka-Demand Of Some Dhātus, Based On Meaning, Following Śābdabodha For Machine
Translation."* For each verb-group (dhātus sharing a meaning) it records, per **kāraka role**, the
**classical citations** attesting that verb taking that kāraka. This is theory-grounded,
citation-backed kāraka gold — the native reference against which SG-SE-013's DCS `deprel` proxy
(~3.93 % coverage) can be measured.

Generated by [`scripts/extract_usha_karaka_gold.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/extract_usha_karaka_gold.py). Model: Opus 4.8 (`claude-opus-4-8[1m]`), H1371 follow-up.

## Provenance

- **Source:** `Concordance/UshaSanka_Ph.D_2014/धातुकारकाकांक्षा.docx` — clean Unicode Word file
  (292 671 Devanagari codepoints; **no OCR** — the bound thesis PDF's text layer is broken, this
  `.docx` is the machine-readable original). Part of the 361 MB **untracked** local trove surveyed
  in [`Concordance/USHA_SANKA_KARAKA_TROVE_REUSE_SURVEY_2026.md`](https://github.com/gasyoun/SanskritGrammar/blob/main/Concordance/USHA_SANKA_KARAKA_TROVE_REUSE_SURVEY_2026.md).
- Because the source is untracked local bulk, the extractor runs **locally**, not in CI (like
  [`sangram/audit/rederive_dcs_numbers.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/sangram/audit/rederive_dcs_numbers.py)); the committed JSON + CSV here are what the repo ships.

## Rights

Usha Sanka's PhD was **published in 2014** (corrected 20-07-2026 — an earlier draft of this README
wrongly called it unpublished). Her **selection and analysis** — this derived kāraka dataset, the
meaning-class model, the role assignments — may therefore be **used, quoted, and republished with
scholarly attribution** (cite her thesis). The classical citations it quotes are public-domain
primary texts (Mahābhārata, Rāmāyaṇa, Purāṇas, Upaniṣads, Gṛhyasūtras, Subhāṣita). The **third-party**
works in the surrounding trove are mostly cleared too: Krishnacharya *Bṛhad-Rūpāvalī* (1924) is
public-domain by age; Abhyankar *Dictionary of Sanskrit Grammar* (1961) and Palsule *Dhātu Concordance*
(1955/1961) are rights-held (usable with attribution); only *Bhāratīya Upasargārtha-Candrikā* (1976) is
still to verify before publishing.

## Caveats

- **Role vocabulary is the author's**, following Śābdabodha: beyond the six Pāṇinian kārakas it adds
  `tādarthya` (purpose), `hetu` (cause) and `anya` (other/miscellaneous). `karman: अकर्मकः` means the
  verb is intransitive (no karman), not a missing value.
- The ~7 % of rows without a `source` are mostly such notes (`अकर्मकः`) or citations whose source the
  original left blank; a handful carry an author query marker `(?)`.
- The author's `चर्चा`/`टिप्पणी` (discussion) prose is captured under each group's `discussion[]`,
  kept **out** of the citation buckets.
- `dhatus[]` are the group's `(). X` headwords; `dhatu_details[]` keeps the full Dhātupāṭha line
  (upadeśa · gaṇa-no · sense · gaṇa · transitivity · iṭ · pada · form) for each.

## Reuse

Primary consumer: **SG-SE-013 `karaka-case`** — as the native, citation-backed reference layer the
DCS `deprel`-proxy kāraka crosstab is measured against (H1371 follow-up handoff). Also feeds any
dhātu-valency / Śābdabodha work and cross-checks the `Dhatu Patha- With Karakas.xlsx` template.
