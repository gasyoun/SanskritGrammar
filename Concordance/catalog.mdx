# Bühler / Knauer / Kochergina — concordance of shared exercise sentences

_Generated by [`scripts/build_catalog.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/build_catalog.py) + [`scripts/render_catalog_mdx.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/render_catalog_mdx.py) from [`scripts/data/matches.json`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/data/matches.json) — do not hand-edit._

Sanskrit grammar textbooks have long borrowed each other's model sentences. This page cross-references the exercise sentences of three textbooks in this repo — [Bühler](https://gasyoun.github.io/SanskritGrammar/grammars/BuhlerLeitfaden_1923/Buhler_Unicode) (German first edition 1878; this repo carries the 1923 Stockholm electronic reprint as a text proxy — the exercises are not known to have been revised between editions, but this has not been verified against the 1878 print), [Knauer](https://gasyoun.github.io/SanskritGrammar/grammars/KnauerFrazy_1908/Frazy-Knauer-03.05.2023) (1908), and [Kochergina](https://gasyoun.github.io/SanskritGrammar/grammars/KocherginaUchebnik_1998/Kochergina_unicode) (1998) — and lists which sentences recur, ordered chronologically by the earliest book each one is attested in.

## Method (first pass — see caveats)

[`scripts/extract_sentences.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/extract_sentences.py) pulls Devanagari- and IAST-script sentence-like runs out of each book's `.mdx` (tagging which script each was originally set in), normalizes whitespace/footnote markers, and computes pairwise matches with `difflib.SequenceMatcher` (similarity ≥ 0.82) within each script pool. [`scripts/build_catalog.py`](https://github.com/gasyoun/SanskritGrammar/blob/main/scripts/build_catalog.py) clusters the pairwise matches (union-find) into single rows and picks a Devanagari exemplar for display when one is available.

**Caveats:** (1) Devanagari matches are the primary signal — all three books share Devanagari unambiguously, whereas IAST runs overlap heavily with glossary/paradigm-table noise (verb-class codes, case labels), so IAST matches are kept only when both sides have ≥4 words. (2) Near-matches below 1.0 are almost all spelling/transcription variants of the same sentence (e.g. *च्छ* vs *छ*), spot-checked by hand — see the review pass in [H327](https://github.com/gasyoun/Uprava/blob/main/handoffs/archive/H327-Sonnet_SanskritGrammar_buhler-knauer-kochergina-concordance_07.07.26.md) (128 raw near-matches classified; only 1 non-Devanagari row survives into the 124-row catalog below, confirming the IAST-noise caveat in (1) is handled by clustering, not just asserted). (3) This is an automated first pass, not a philological edition — it will miss paraphrased reuse (same content, different wording) and may include coincidental short-sentence collisions; treat single-book-pair rows with short text as lower confidence than the all-three-books rows. (4) Sentence numbering below is catalog-only (`C0001`…) — the source books' own numbering is untouched.

**124 shared-sentence clusters found:** 7 attested in all three books, 79 shared only between Bühler and Knauer, 33 only between Bühler and Kochergina, 5 only between Knauer and Kochergina.
