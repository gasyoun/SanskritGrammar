# CH7_PLAN — Гл. 7 «Грамматический указатель корней по модели Зализняка» (Часть II, капстоун)

_Created: 16-07-2026 · Last updated: 16-07-2026_

План последней главы Части II — прикладной кульминации. [H1037](https://github.com/gasyoun/Uprava/blob/main/handoffs/H1037-Opus_SanskritGrammar_m03-ch7-zaliznyak-index_16.07.26.md),
Opus 4.8 (`claude-opus-4-8[1m]`).

## 1. Тезис главы

«Дальнейшее направление работы» Заключения 2014 г. — прикладной **грамматический указатель корней
по модели Зализняка** — к 2026 г. выполнимо. Зализняк связывал каждое слово с полной парадигмой
одним компактным кодом; та же идея, приложенная к глагольному корню, даёт **структурированные
грамматические данные**: код `ряд · seṭ/aniṭ · полноизменяемость · схема ударения` → полное
описание корня. Глава показывает готовую схему кода на кросс-уоке и честно называет незакрытое
(прежде всего — ударение, которого нет в корпусе, Гл. 6).

## 3. Структура (черновик)

- **Введение:** «будущая работа» 2014 г. как готовый ресурс; код Зализняка → парадигма; структурные данные, не пособие.
- **§7.1 Схема кода:** модель Зализняка (помета/тип/ударение/флаги, номинальный аналог — ссылка на A56/ZALIZNYAK_INDEX); глагольно-корневой код = ряд (18) + seṭ/aniṭ (308/287/176) + полноизменяемость (429/424/22); охват 876/930 корней; уверенность ряда (446/410/20).
- **§7.2 Ударение — трудная ось:** корпус акцента не несёт (Гл. 6); ударение восстанавливается 18 правилами по Уитни (схемы a–f, сверка на 19 клетках) и эмпирически — из акцентуированной Ригведы (VedaWeb). Честный предел: покрытие частичное, схема ударения не для всех корней.
- **Выводы:** указатель реализует «будущую работу» 2014 г.; жёсткая рамка (930 корней кросс-уока); замыкает Часть II.

## 4. Антисаламная граница

- **Цитировать:** A56 (номинальный грамматический токен-индекс Зализняка, kosha) — методический
  сибл; разбор трёх работ Зализняка (1975/1978/2004) — Гл. 3 §3.3 и ZALIZNIAK_1975_1978_2004_COMPARISON
  (H357), **не повторять**; акцентная слепота I/VI, IV/пассив — A06/A32.
- **Собственный вклад:** сведение ряд+seṭ/aniṭ+полноизменяемость+ударение в единый корневой код на
  кросс-уоке — прикладной указатель, которого нет ни в одном отдельном активе.
- **Оговорка о верификации:** точный список разрядов Зализняка сверить с оригиналом 1975/2010 г. до
  публикационных утверждений (см. предупреждение в ZALIZNYAK_INDEX.md).
