## Что именно слито / переписано / сохранено

### Слито (2026-данные перенесены в основной текст)

1. **§3.2 -- три работы А. А. Зализняка и постатейная атрибуция.** В основной текст внесён
   разбор: «по Зализняку» -- отсылка неоднозначная, так как санскрит описан трижды разным
   аппаратом (статья 1975 г. / очерк 1978 г. / конспект 2004 г.), полной грамматики среди
   трёх нет (это Уитни 1889 г.). Атрибуция по-механизмная: позиции 1/2/3 + исчисление
   типов I--IV → **1975**; двухуровневая запись I/II → **1978**; диахрония + и.-е.
   соответствия → **2004**. Ранее это ядро жило только в сноске `[^ed24c]` к §2.4. Сноска
   `[^ed3b]`.
2. **§3.2 -- линия Уитни → Зализняк → Толчельников** («описать → абстрагировать →
   породить»): «Талмуд» развивает как раз линию 1975 г., от которой сам Зализняк в
   1978/2004 гг. отказался. Тип «Талмуда» (s/a/v) ≠ тип Зализняка 1975 г. (I--IV) --
   разные оси, сходно поименованные.
3. **§3.2 -- совпадение инвентаря рядов** «Талмуда» и Зализняка (покорневая сверка 876
   корней по [crosswalk-CSV](https://github.com/gasyoun/SanskritGrammar/blob/main/TolchelnikovTalmud_2026/data/morphoclass_crosswalk_1975_2014_2026.csv))
   + снятие мнимых рядов `I₀/M₀/N₀/R₀/U₀` (баг базы `/z/`, [issue #50](https://github.com/gasyoun/SanskritGrammar/issues/50)).

### Переписано (аргументация 2014 г. заменена, без цитат)

4. **Открытие главы -- terminology-абзац** (виза автора есть): морфонология ⊃ морфонемика,
   запись корней = морфонемная абстракция, именуемая «морфонологической записью».
   Развёрнут из оговорки Введения (бывш. строка ~321→323); во Введении оставлена краткая
   отсылка. Сноска `[^ed3a]`.
5. **Выводы по третьей главе -- переписаны начисто**, без цитирования формулировок 2014 г.
   Аргумент: спектр способов записи (заглавные буквы → алгебраический метаязык ← анубандхи
   Панини); обособленная линия абстрактной записи Зализняка с постатейной атрибуцией;
   продолжение её «Талмудом»; трудоёмкость сравнения списков (пример √*hū*); направление к
   конкордансу (§3.4) через XML-связки. Сноска `[^ed3d]`.

### Решения (с обоснованием)

6. **Таблица 9 (Зализняк ↔ Моргенрот) -- третья колонка НЕ вводится** (DO#4). Инвентарь
   рядов «Талмуда» совпадает с зализняковским 1975 г., отдельный столбец продублировал бы
   левую часть; вклад «Талмуда» -- порождающий механизм над рядами, не новый их перечень.
   Прозой + сноска `[^ed3c]`.

### Сохранено (НЕ значит «идеально»)

- **§3.1 (структура корня) и §3.3/§3.4** содержательно не переписывались: 2026-данные их
  тезисы не меняют. Их *проза и аппарат* -- предмет отдельного стилевого прохода:
  RWS-проход дал по Гл. 3 **269 замечаний (129 major/critical), максимум по книге**
  ([RWS_REPORT.md](https://github.com/gasyoun/SanskritGrammar/blob/main/GasunsDhatu_2014/revision-2026/RWS_REPORT.md)) --
  IAST при первом упоминании, authority chain, ссылки на сутры. Переписывание прозы вне
  объёма H378 (как §2.1–§2.2 для Гл. 2).
- **Вычислительные результаты не пересчитывались** (Таблицы 6–9, verified H246/H328;
  сноски `[^edt1]`/`[^edt4]`/`[^edt5]`/`[^edl9a]`).
- Нумерация ВАК-положений не тронута; аблаутный Ряд A–N не смешан с варгами (DO NOT).
