Занятие X — основы на -ā (HK-14) + имперфект (HK-15)
Годится в печатную рукопись как написано? (заметка — для точечной правки)

Заметки

(1) «-ā женского рода» — правило класса, не звука (корневые ā-основы вне правила); (2) имперфект «обычно повествование» — верно для классической прозы, но скрывает расслоение по эпохе.

Чтения

aśvā aśrāmyan bhūmāv apataṃś ca (Bühler XII = Кочергина X, C0008) · udyāne patadbhyo vihagebhyo dhānyaṃ kiratīḥ kanyā apaśyam (Bühler XXIII).

Дриллы + ключ

Склонение senā: senā / senām / senayā / senāyai / senāyāḥ / senāyāḥ / senāyām / sene. Презенс→имперфект: abhavat, agacchat, avadat, apaśyat, avasat.

Склонение nadī: nadī / nadīm / nadyā / nadyai / nadyāḥ / nadyāḥ / nadyām / nadi. Род: nadī ж · rathī м · senānī м · vadhū ж · svayaṃbhū м.
