Урок 5.

<u>Каузатив.</u>

**§ 42.** \"Каузативная форма глагола передаёт идею того, что некое лицо или предмет побуждает другое лицо или предмет совершать действие или испытывать состояние, обозначенное глаголом\". (Dr. Kielhorn\'s Grammar § 416); например, gm! *идти*, gCDit *\'он идёт\'*; gmyit *\'он побуждает идти\'*; Az! *есть, принимать пищу*; Aîait *\'он ест\'*; Aazyit *\'он кормит, заставляет есть\'*.

**§ 43.** При каузативе *субъект* глагола в своём исходном значении ставится в творительный падеж, *объект* остаётся без изменений. Например:

+------------------------------+---+---------------------------------------------+
|                              |   |                                             |

**§ 44.** [^1] В случае каузатива с глаголами, подразумевающими *\'движение\'*, *\'знание\'* или *\'сообщение\'*, или какое-либо *\'принятие пищи\'*, и для глаголов со сходными значениями, а также для глаголов, имеющих в качестве объекта литературное произведение, и для непереходных глаголов, *субъект* глагола в своём исходном значении ставится в винительный падеж, *объект* остаётся неизменным. **§ 45.** У вышеприведённого правила есть несколько важных исключений и встречных исключений:

a)  [^3] Каузативы от глаголов nI *\'вести\'* и vhœ *\'нести, везти\'* управляют *творительным*, а не винительным падежом, например:

```rst-table
+---------------------------+---+----------------------------------------+
| -&Tyae -ar&lt; nyit vhit va |   | -&Tyen -ar&lt; nayyit vahyit va ( S.K. ) |
|                           |   |                                        |
| *Слуга несёт* груз*.*     |   | *(Он) побуждает слугу нести* груз*.*   |
+---------------------------+---+----------------------------------------+
```

Но глагол vhœ в тех случаях, когда субъектом глагола выступает слово со значением *\'лицо, направляющее движение чего-либо\'*, подчиняется общему правилу, например:

```rst-table
+---------------------------+---+---------------------------------------------+
| vaha rw&lt; vhiNt           |   | sUtae vahan! rw&lt; vahyit ( S.K. )           |
|                           |   |                                             |
| *Лошади везут колесницу.* |   | *Кучер заставляет лошадей везти колесницу.* |

|                           |   |                                             |
+---------------------------+---+---------------------------------------------+
| vhiNt yvan! blIvdaR&gt;     |   | vahyit yvan! blIvdaRn! ( Mbh. )             |
|                           |   |                                             |

+---------------------------+---+---------------------------------------------+
```

b)  [^4] Каузативы от глаголов Adœ и oadœ -- *\'есть, принимать пищу\'* управляют творительным падежом, например:

```rst-table
+--------------------------+---+--------------------------------------+

|                          |   |                                      |
| *Мальчик ест пищу.*      |   | *(Он) кормит мальчика пищей.*        |
+--------------------------+---+--------------------------------------+
```

c)  [^5] Глагол -\]! в том случае, когда он не имеет значения ih---sa -- *\'причинение вреда живому существу\'*, управляет творительным падежом, например: -\]yit ip&lt;fI--- devdÄ&gt;, -\]yit ip&lt;fI--- devadÄen; но -\]yiNt yvan! blIvdaR&gt;, -\]yit blIvdaRn! yvan! (Mbh. ).

**§ 46.** В параграфе § 44 под *непереходными* подразумевались глаголы, по своей природе не способные управлять другим дополнением, кроме как со значением \'время\', \'место\' &c., а не те, которые, будучи переходными, иногда, - по воле говорящего или когда их значение достаточно очевидно, - могут быть употреблены как непереходные. Так: ik&lt;kr&gt; pcit. Здесь глагол pcit, будучи переходным, употребляется без дополнения, потому как он достаточно прост для понимания. Поэтому ik&lt;kre\[ pacyit, а не ik&lt;kr&lt;, но masmasyit devdÄ&lt;.

**§ 47.** [^7] При образовании пассивной конструкции с каузатив ным глаголом *прямое* дополнение при каузативе, являющееся *субъектом* глагола в его исходном значении, ставится в именительный падеж, другое дополнение остаётся неизменным. Например:

+:------------------------:+:-:+:---------------------------------------:+:-:+:-----------------------------------:+

Но в случае глаголов, выражающих *\'знание\'*, *\'принятие пищи\'*, и для глаголов, имеющих в качестве объекта литературное произведение, прямое дополнения ставится в именительный падеж, косвенное дополнение -- в винительный или *наоборот.* Например:

**§ 48.** Формы каузатива у глаголов десятого класса совпадают с исходными формами, а значение определяется контекстом. Так:

ramae xn&lt; caeryit -- *\'Рама ворует богатство\'*;

ramae gaeivNden xn&lt; caeryit -- *\'Рама побуждает Говинду воровать богатство\'*.

В последнем предложении глагол имеет каузативное значение.

**§ 49.** В отношении глаголов, управляющих двумя дополнениями в винительном падеже, правила, приведённые в параграфах § 43 и § 44, также применимы, т.е. глаголы, означающие *\'движение\'* и т.д., управляют винительным падежом исходного субъекта, а другие -- творительным падежом на условиях, приведённых в параграфе § 45.
